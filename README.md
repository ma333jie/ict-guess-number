# 猜数字大挑战 (Guess the Number)

一个简约、色彩鲜艳且充满现代感的网页猜数字游戏。

## 🎮 游戏简介
这是一个经典的猜数字游戏，系统会随机生成一个 1 到 100 之间的数字，玩家需要在有限的次数内猜出这个数字。

## ✨ 核心特性
- **现代感 UI**: 使用了高饱和度的色彩搭配和流畅的 CSS 动画。
- **即时反馈**: 每次猜测都会提示“太大了”或“太小了”，并实时更新剩余机会。
- **响应式设计**: 支持桌面端和移动端浏览。
- **趣味交互**: 包含错误输入的“震动”特效。

## 🚀 快速开始
只需在浏览器中打开 `index.html` 即可开始游戏。

### 本地开发预览
如果你想启动一个简单的本地服务器进行预览，可以使用 Python：
```bash
python3 -m http.server 8000
```
然后在浏览器访问 `http://localhost:8000`。

## 🛠️ 技术栈
- **HTML5**: 页面结构
- **CSS3**: 样式与动画 (使用 Flexbox 布局)
- **JavaScript (Vanilla JS)**: 游戏逻辑处理

## 🌐 部署说明
本项目非常适合部署在 **GitHub Pages**：
1. 将代码推送至 GitHub 仓库。
2. 进入仓库设置 (Settings) -> Pages。
3. 选择 `main` 分支并保存。
4. 几分钟后，你的游戏就可以通过 GitHub 提供的链接在线访问了。
